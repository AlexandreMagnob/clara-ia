# Tiers de Leads e Delegação

## Classificação de Tiers

Os leads são classificados em tiers (1–5) pela automação após preencher o formulário.

| Tier | Perfil | Responsável |
|---|---|---|
| 1 | Alto potencial | Vendedor humano (Meetime) |
| 2 | Alto potencial | Vendedor humano (Meetime) |
| 3 | Potencial médio | Vendedor humano (Meetime) |
| 3.1 | Potencial médio-baixo | Vendedor humano (SDR) |
| 3.2 | Potencial médio-baixo | Vendedor humano (SDR) |
| 4 | Potencial baixo | Clara IA |
| 5 | Potencial baixo | Clara IA |

## Fluxo de Delegação

1. Lead preenche formulário de entrada
2. Automação (N8N) mapeia o tier do lead
3. Tier 1–3: delegado para vendedores via Meetime
4. Tier 4–5: enviado para Clara IA iniciar prospecção

## Objetivo da Clara nos Tiers 4–5

- Prospectar e engajar o lead
- Converter o interesse
- Marcar reunião/chamada para o closer fechar a venda
