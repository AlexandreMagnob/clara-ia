# Prompts em Produção

Esta pasta contém os system prompts e instruções da Clara atualmente em produção.

## Como usar

- Cada arquivo `.md` ou `.txt` representa um prompt ou bloco de instrução da Clara
- Ao atualizar um prompt no N8N, copiar a versão anterior para `../historico/` antes de sobrescrever
- Nomear os arquivos de forma descritiva: `system-prompt-principal.md`, `instrucoes-prospeccao.md`, etc.

## Convenção de nomenclatura

```
YYYY-MM-DD_nome-descritivo.md
```

Exemplo: `2026-05-14_system-prompt-principal.md`
